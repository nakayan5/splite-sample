sqlite tutorial

## command

```
<!-- dbを指定 -->
sqlite> .database

<!-- table一覧 -->
sqlite> .table

<!-- schemaを指定 -->
sqlite> .schema <name>

<!-- table構造取得 -->
sqlite> PRAGMA table_info(<name>);
```
